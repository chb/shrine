1.2.5 Previous Queries View

When results from a successful query are returned in the Query Status pane in the lower right corner of the SHRINE Web Client, an entry also appears in the Previous Queries pane in the lower left corner.

This entry is comprised of:

  • an expandable file icon, with plus sign [+] on the left
  • the query name (term(s)@hh:mm:ss)
  • the date the query was run ([mm-dd-yyyy])
  • your username [xxnn]

To View the Results of a Previous Query

  1. Click the plus sign [+] of the expandable file icon. A revolving progress icon may appear. An entry will appear below the query name entry. This entry is comprised of:
    • an expandable file icon, with plus sign [+] on the left
    • Results of (term(s)@hh:mm:ss)[mm-dd-yyyy][xxnn]
  2. Click the plus sign [+] of the expandable file icon. A revolving progress icon may appear. The results of the selected previous query appears. Each result is comprised of:
    • A patient number icon
    • Text reading: Patient Count from <name of institution> - Fewer than 10 patients or <number> Â±3 patients

To Run a Previous Query Again

Drag the icon of a Previous Query to any Group pane of the Query Tool area in the upper right and release. This populates the term(s) in the Group pane(s) in precisely the same manner as in the original query. Now, as with any query, select an Approved Query Topic, click the Run Query button, and click OK in the Run Query dialog.
