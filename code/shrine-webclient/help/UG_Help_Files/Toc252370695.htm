1.3.2 Logging In to SHRINE

After your registration status is approved at your local institution, access the SHRINE Query Tool using the address, username, and password provided to you.

  1. Point your browser to the address identified when you registered for SHRINE.
    Note: You must be behind your hospital's firewall to view the login page.
  2. Enter your username and password.
  3. Use the SHRINE Host pull-down list to select the version of SHRINE available at your institution.
  4. Click Login.
    You see a Terms of Access Agreement that you must review and accept before you can use the Query Tool.
  5. To use the SHRINE system, click Yes, I agree.
    You see the SHRINE system in your browser window.

Continue to Submitting a Query Topic Request to find out how to access the SHRINE Query Tool.
