This User Guide, the first section of the SHRINE Help Site, describes the basics of using the SHRINE Query Tool:

  • Register
  • Request Query Topic approval (Instructor or above)
    • Recruit/Sponsor a Fellow (Instructor or above)
    • Participate in Query Topic (Fellow)
  • Construct a query (all approved users)
