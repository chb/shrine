1.2.2 Terms View

Use the patient Terms view, on the left side of the SHRINE window, to navigate (browse) or find (search) the medical terms on which you choose to query. The full range of terms available for you to query is listed in this view, grouped into four main criteria:

  • Demographics
  • Diagnoses
  • Lab Tests
  • Medications

Each of the main criteria contains multiple categories that are displayed in subfolders, which contain more-specific terms by which you can refine your query. As you navigate through the subfolders and locate a term or category that you choose to include in your query, drag that entry to the Query Tool and drop it in a Group column to add it to your query criteria. You can select term folders, subfolders, or discrete terms to add to your query.

If a term category maps to more than 120 specific terms, you cannot use that category in the Query Tool. You must expand it and select a more specific term to use. Otherwise the query would take too long to process.

Fields and functions in the Terms view include:

  • Navigate Terms tab – Default tab of the Terms view. Use the Expand (+) and Compress (-) buttons to browse through available characteristics on which to query.
  • Find Terms tab – To search for specific characteristics on which to query, click the Find Terms tab on the Terms view. You can search by using the following criteria:
    • Search by Name – This is the default tab of the Find Terms view.
      To search by Name:
      1. Click the top pull-down list to select the logical search operator. Options include Contains, Exact, Left (starts with), or Right (ends with).
      2. Enter your search string in the text field beside the operator pull-down list.
      3. Click the Category pull-down list to select which category of data and select the SHRINE option.
      4. Click Find.
    • Search by Codes – Click this tab to search criteria for specific coding system terms.
      To search by Code:
      1. Use the Select a Coding System pull-down list to select the specific code type for which you choose to search.
      2. Enter the term in the text field above the list.
      3. Click Find.
      Coding systems on which you can search include:
      • Demographics (Marital)
      • Demographics (Sex)
      • Demographics (Language)
      • Demographics (Age)
      • Demographics (Race)
      • Lab Tests
      • Diagnoses (ICD9)
      • Medications (RxNorm)
