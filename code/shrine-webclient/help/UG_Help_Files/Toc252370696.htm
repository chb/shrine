1.4 Submitting a Query Topic Request

Before you can construct a query to determine patient populations, you must submit a request through the Data Steward application and define a specific topic for which you choose to query patient data. The Data Steward administrator reviews all topics before granting approval and subsequently monitors query activities for all approved topics to ensure that no inappropriate use of the SHRINE system occurs.

You can request multiple Query Topics to support different selection criteria of patient cohorts.

To submit a Query Topic request:

  1. In a browser window, go to the Data Steward home page.
    Note that you also can access the Data Steward application from the SHRINE system. In the Query Tool pane, click Request New Topic at the bottom-left of the tool.
  2. In the Data Steward window, complete the following information:
    • Enter a Query Name for your topic.
    • In the Query Intent field, describe the research topic for which you choose to explore potential cohorts.
    • Use the Reason for Query pull-down list to select the reason for which you choose to explore this specific topic. If you select a reason of "Other," add an explanation for your reason in the text field below the pull-down.
    • If you are collaborating with a non-SHRINE associate, use the pull-down list to select Yes. Then add an explanation in the text field below that pull-down.
      Note: Collaborations with individuals or entities outside of the participating SHRINE institutions, including academia and industry, currently are prohibited.
  3. Click Submit Request.
    The Data Steward administrator will review your Query Topic to determine if it constitutes appropriate use of SHRINE data. If more information is needed, the administrator will contact you. We try to process all requests within 48 hours.
    You can check the Data Steward application to view the status of your request as listed in the "Your active requests" field at the bottom of the window. When your request is approved, you now have access to SHRINE data.
  4. Click Logout to exit the Data Steward application.
