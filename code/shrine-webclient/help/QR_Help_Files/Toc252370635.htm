3. SHRINE Data Reference

2.1.3 Sample Query Topic

Following is a sample of how you might describe your query request to the Data Steward:

Query Topic: IBD and Colon Cancer

Query Intent: I am interested in the association between inflammatory bowel disease and colon cancer. Literature shows that patients diagnosed with IBD are at higher risk for cancers of the colon, but it is unclear why. I would like to find the largest possible sample size of IBD patients who have a co-morbid diagnosis of colon cancer to see if it is possible to discern patterns by demographic characterization (e.g., age, gender, ethnic identity).

Reason for Query: Hypothesis Generation
