2.2.2 Constraints

At the top of each group are three criteria by which you can limit all query terms within that group:

  • Dates – Set a date range to query terms associated with patients for that period.
    For example, a constraint setting of Diagnosis of Asthma for <date range> returns all patients diagnosed with asthma within the specified date range.
  • Occurs > 0x – Set the number of times that patients must be associated with the query term.
    For example, a constraint setting of Diagnosis of Fracture of ankle Occurs > 2x returns all patients diagnosed with a broken ankle at least three times.
  • Exclude – Exclude patients meeting the specified term or terms.
    For example, exclude all patients 35 years old returns all patients who are not 35 years old.

To remove a constraint, click the constraint again and remove the check box from the value that you set. For an exclusion constraint, simply click Exclude again to remove it.

Note: Not all terms accept all constraints. For example, a demographic gender term of Female does not accept a date range constraint.
