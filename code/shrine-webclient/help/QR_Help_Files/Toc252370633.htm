3. SHRINE Data Reference

2.1 Query Topics

Query Topic requests are submitted through the Data Steward application You must submit a request and have a topic approved to run a query in SHRINE.

2.1.1 Data Steward

The Data Steward administrator decides whether to approve query topics. After a topic is approved, the administrator also regularly reviews use of the Query Tool to ensure that no suspicious or unapproved examination of data is performed.

You can access the Data Steward application from within the SHRINE Query Tool. Click Request New Topic in the tool view to open the application in a new window, and log in by using your eCommons account credentials.

Or, access the application directly through this link.
