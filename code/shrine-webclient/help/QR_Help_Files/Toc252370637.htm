2.2 Query Construction

Two icons at the top-right corner of the Query Tool view provide the following options:

  • Show Options icon – Click this icon at the top-right corner of the view to Select the Maximum Time to Wait for XML Response (in seconds). This value is the length of time that SHRINE will continue to process a query. If a query does not process completely within [period], only those results that completed processing within an institution are returned.
  • Toggle Zoom icon – Click the Toggle Zoom icon at the top-right corner of the view to toggle between expanded and compressed view sizes.

2.2.1 Tool Logic

You can add terms within a Group to apply OR logic, and use multiple Groups to define your selection by using AND logic. Tool options function as follows:

  • Query Name – This field remains blank. You cannot edit this field.
  • Group columns – Query groups represent a set of patients defined by the terms within the group and any constraints applied to that group. Drag terms into one or more Group fields to narrow the patient cohort for which you are querying.
  • Constrain group options – Define criteria by which you can limit all terms within a group:
    • Dates – Set a date range within which the terms associated with patients must fall.
    • Occurs > 0x – Set the number of occurrences for which patients within the query group must be associated with a term.
    • Exclude – Exclude all patients that meet the criteria of the query group.
    Click the constraint a second time to remove it.
    Note: Not all terms accept all constraints.
  • Term right-click menu – Right-click a term to set a specific value by which to query a term, or to delete the term from the query group.
  • Select a Topic pull-down list – Select the query topic to which this query relates, as approved by the Data Steward.
  • Info button – View the description of that topic as provided to the Data Steward.
  • Request New Topic button – Access the Data Steward in a new window.
  • Run Query button – Start the query process.
  • New Query button – Clear the Query Tool fields and settings and start a new query.
  • New Group button – Add additional terms and values to your query if needed.
  • Group page controls – Navigate to additional groups if you add new groups.
